# Operations Manuals & User Guides

## 1. Customer User Guide
1. **Onboarding**: Go to `/register` to create a customer node. Use `/otp` to verify status.
2. **Browsing**: Query the search console to locate restaurants and dishes.
3. **Cart**: Add food items to cart and click checkout.
4. **Loyalty Wallet**: Go to the **Wallet & Loyalty** page to add cash, review points balance, and claim vouchers.

---

## 2. Administrator Operations Manual
1. **Security Operations**: Monitor CPU load, V8 heap usage, and database connections. Use the **AES-256-GCM sandbox** to encrypt sensitive data.
2. **QA & Testing**: Trigger Jest test suite simulations and execute database load benchmarks.
3. **AI Fraud Detection**: Monitor the Threat Level index gauge. Trigger simulations (Coupon abuse, Fake reviews, Card velocity, Bots) and block accounts.
4. **DevOps & Infrastructure**: Inspect EKS deployment manifests and run Terraform plan simulations.

---

## 3. Restaurant Partner Manual
1. **Outlet Onboarding**: Log in as a restaurant owner to access `outlets/onboard`. Enter location coordinates, descriptions, and cover image URLs.
2. **Menu Creation**: Go to the Outlets Manager to add category folders and create dishes. Configure Veg/Non-Veg details, calories, and initial stock variables.

---

## 4. Delivery Driver Operations Manual
1. **Active Map Console**: Displays active customer orders, customer location markers, and optimal route lines.
2. **ETA Updates**: Calculates real-time delivery durations based on spatial distances, weather constraints, and traffic data.
